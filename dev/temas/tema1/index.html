Álgebra y Ecuaciones

1. Resolver un sistema sobredeterminado por mínimos cuadrados

using LinearAlgebra

# Sistema sobredeterminado Ax ≈ b
A = [1 1; 1 2; 1 3]
b = [2.0, 2.5, 3.5]

# Solución por mínimos cuadrados
x = A \ b
2-element Vector{Float64}:
 1.166666666666668
 0.7499999999999997

julia using JuMP

model = Model() @variable(model, x >= 0)
